Report Center

报告中心

面向月度物流、库存和费用验收汇报，把预览、抬头 Icon 和 PDF 导出集中到一个入口。

报告生成器

未上传，使用文字抬头

报告类型说明

月度经营总报告汇总物流、库存、费用验收和关键风险，适合月度邮件正文或管理层汇报。
物流报告聚焦订单履约、运输类型、需求国家、重量体积和费用趋势。
库存报告聚焦库存规模、货值、DOS、长期未使用和主数据完整度。
验收报告聚焦发票匹配、已验收金额、待验收金额和异常原因。
超期物料报告聚焦 DOS 超期和长期未使用物料，适合月度风险跟踪。
数据质量检查用于检查缺图片、缺价格、缺入库时间、未匹配物流或发票。

📦 月度经营总报告

用于整理当月物流、库存、货值、图片与费用验收状态，适合月度汇报前生成 PDF。

  • 包含库存清单、图片、物流数据、发票验收状态
  • 按当前仓库或全部仓库导出

🚚 月度物流报告

单独整理当月运单状态、运输类型、国家/代表处要货、重量体积和费用趋势。

  • 适合发给物流相关同事
  • 聚焦订单履约和运输费用

📋 月度库存报告

单独整理库存总货值、DOS 超期、长期未使用、图片和主数据质量。

  • 适合每周/月度库存复盘
  • 聚焦库存健康和超期风险

📊 适用产品库存清单

按 Category 2 / 适用产品导出正式库存清单报告，适合单独分析 Phone、Tablet、Wearable 等产品维度。

  • 带统计总览、库存货值、PCS 和明细清单
  • 可按报告生成器里的统计范围选择仓库

✅ 月度验收报告

单独整理物流费用发票验收、匹配率、待验收金额和异常提醒。

  • 适合费用验收和对账沟通
  • 聚焦已验收、待验收和未匹配 STT

⏱ 月度超期物料报告

聚焦 DOS 超期和长期未使用物料，导出 Excel 会带统计总览、明细清单和匹配图片。

  • 图表同时展示货值与 PCS
  • 清单支持筛选、排序、分页和图片预览

🔎 数据质量检查

报告前建议先检查三个关键风险：库存表是否覆盖成功、维护表是否有缺失 BOM、图片是否能按 BOM Code 匹配。

  • 供应商库存：以新上传为准覆盖
  • 维护表：以 Product Number 映射 BOM / 价格 / 阈值

建议汇报流程

1. 刷新库存上传供应商库存；维护表只有变化时再覆盖。
2. 检查图片图片文件名保持 BOM Code；上传错了可一键清理后重传。
3. 看风险进入库存看板，重点看 DOS 超期和长期未使用。
4. 导出报告生成月度预览，确认后导出 PDF；超期明细可另导出 Excel。
报告预览 建议优先导出 HTML 长报告，邮件正文可用长图，PDF 用于归档
